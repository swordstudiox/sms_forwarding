系统设置

管理账号、设备时间、局域网域名与维护（重启 / 配置备份 / 固件升级）

管理账号
首次使用请立即修改默认密码！默认：admin / admin123
设备时间

用于设备对时与本地时间显示；保号倒计时依赖对时，修改 NTP 需重启生效

局域网域名 (mDNS)

通过 http://主机名.local 访问设备；多台设备请设置不同主机名，避免互相顶替。仅限小写字母、数字和连字符，保存后约 1 秒生效，无需重启

NET 指示灯

拨动开关即时生效（临时下发，不写配置）；点“保存”才永久保存，设备重启 / 模组重新初始化后按保存值恢复

设备维护

恢复出厂将清空所有配置并重启（WiFi 凭据为固件内置，不受影响）

配置备份 / 恢复

默认导出会隐藏 WiFi、账号、SMTP 和推送密钥；完整导出含凭据，请妥善保管。恢复：粘贴备份内容后点导入

固件升级 (OTA)

上传编译生成的 .bin 固件；填写 SHA-256 时设备会校验固件内容，不匹配则拒绝启用；升级中请勿断电，约需 30-60 秒
